# ZenPaws Web Storefront & Interactive Pipeline Prototype

This repository contains the interactive prototype for the **ZenPaws** brand. It is ready for hosting on Netlify, Vercel, or GitHub Pages.

## 🚀 One-Click Deployment Guide

### Option 1: Vercel (Recommended)
1. Install Vercel CLI: `npm install -g vercel`
2. Run `vercel` in this directory.
3. Follow the CLI prompts to deploy.

### Option 2: Netlify Drag-and-Drop
1. Visit [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop this folder (`web-prototype`) into the browser upload box.
3. Your site will be online in seconds.

## 📁 Directory Structure
*   `index.html` - The storefront user interface containing the products and the interactive pipeline visualizer.
*   `style.css` - Custom premium styling (fonts, colors, mobile responsive layout).
*   `app.js` - Cart logic and the simulation state-machine.
*   `logo.jpg` - Brand identity logo.
